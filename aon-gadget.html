AON Gadgets – E-commerce Platform

AON Gadgets is a high-performance e-commerce web application developed in collaboration with two other frontend developers. Built using pure HTML, CSS, and JavaScript, the platform provides a streamlined shopping experience for modern technology enthusiasts.

This project served as a pivotal learning milestone, marking my first deep dive into professional version control. By utilizing Git and GitHub, our team managed collaborative workflows, resolved merge conflicts, and maintained a clean codebase throughout the development lifecycle.

A core technical achievement was engineering a persistent cart system by storing complex data structures and dictionaries directly in the browser's Local Storage to maintain user data across sessions.

Project Features

HTML5 • CSS3 • JavaScript • Git

One of the technical highlights I spearheaded was the implementation of the product management logic. I focused on using JavaScript to dynamically render gadget lists and manage state changes, ensuring that the interface responded instantly to user actions without the need for external frameworks.

By mastering the storage of dictionaries in Local Storage, I ensured that user preferences and shopping carts remained intact even after page refreshes. This project demonstrates my ability to work effectively in a developer team and build functional, data-driven applications from the ground up.
